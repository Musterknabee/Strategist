# ADR-0095: Chronic-Origin Restoration Audit Overlay

## Status
Accepted

## Context
Chronic-origin work that converged back into the standard restoration path carried provenance but that provenance did not yet influence restoration intensity or drift policy.

## Decision
Introduce a typed restoration-audit overlay for chronic-origin items and derive a provenance-aware drift policy from that overlay.

## Consequences
Chronic provenance now changes live restoration and drift posture instead of remaining passive metadata.
