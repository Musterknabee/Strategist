# ADR-0078: Operator Recurrence Tribunal Lane

## Status
Accepted

## Context
Chronic instability packets need a dedicated routing surface that distinguishes supervisor recurrence review from tribunal-grade recurrence review.

## Decision
Introduce `oracle_operator_recurrence_tribunal_lane/v1` as the typed routing lane for recurrent and chronic reopened remediation work.

## Consequences
Structurally unstable work receives durable, reviewable routing semantics with explicit authority, destination, and remediation directive.
